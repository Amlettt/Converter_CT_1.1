Инструкция

Подготовь оргео правильно:


1. Используй в названии групп только русские буквы!

2. Можно использовать буквы большие/маленькие/комбинацию.

3. Связку/группу создавать как эстафету!

4. Поле "номер чипа" оставить!


Примеры названий:

1КЛ_М_16-21, 5КЛ_Ж - личка

1кл_св_м_14-15, 3кл_св_см - связка

2Кл_Гр_Ж_10-11 - группа
